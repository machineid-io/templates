# MachineID.io Templates

Copy-paste examples for using MachineID.io with popular frameworks.

| Framework / Type       | Repo                                                                                         | Description                                                                                      |
| ---------------------- | -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| **Python Starter**     | [machineid-python-starter](https://github.com/machineid-io/machineid-python-starter)         | Universal base template for agents, workers, scripts, and automation (register + validate cycle) |
| **CrewAI**             | [crewai-machineid-template](https://github.com/machineid-io/crewai-machineid-template)       | Enforced fleet limits for CrewAI agents *(coming soon)*                                          |
| **OpenAI Swarm**       | [swarm-machineid-template](https://github.com/machineid-io/swarm-machineid-template)         | Prevent runaway Swarm costs                                                                      |
| **LangChain / LCEL**   | [langchain-machineid-template](https://github.com/machineid-io/langchain-machineid-template) | Self-registration + validation in LCEL                                                           |
| *(more coming weekly)* | —                                                                                            | —                                                                                                |
